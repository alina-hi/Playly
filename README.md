# Playly

**Социальная сеть для родителей с интерактивной картой детских площадок**

Playly — это веб-платформа, которая помогает родителям находить, оценивать и делиться информацией о детских площадках в своем городе. Приложение объединяет интерактивную карту, систему оценок и социальные функции для создания сообщества ответственных родителей.

## Функции 

- **Информация о детях** — возможность указания в профиле информации о детях (возраст, интересы)
- **Редактирование профиля** — изменение личных данных и предпочтений
- **Умный поиск** — поиск площадок по аттракционам, возрасту, удобствам
- **Фильтрация объектов** — фильтрация по типу, оснащению и ключевым критериям оценки

## Роли пользователей

| Роль | Описание | Основные возможности |
|------|----------|---------------------|
| **Родитель** | Основной пользователь приложения, ищет безопасные места для досуга с детьми | Просмотр карты площадок, оценка объектов по критериям, управление профилем с информацией о детях |
| **Модератор** | Сотрудник платформы, отвечающий за качество и безопасность контента | Редактирование информации о площадках, модерация отзывов, блокировка нарушителей |
| **Партнер** | Представитель девелоперской компании или локального бизнеса | Получение обратной связи от пользователей, улучшение объектов на основе оценок |

## Функциональные возможности

### Для всех пользователей:
- Регистрация и создание профиля
- Просмотр интерактивной карты детских площадок
- Фильтрация объектов по типу и оснащению
- Просмотр детальной информации о площадках

### Для родителей:
- Оценка площадок по структурированным критериям (безопасность, чистота, оборудование)
- Указание информации о детях в профиле (возраст, интересы)
- Участие в чатах площадок
- Организация мероприятий с другими родителями

### Для модераторов:
- Верификация пользовательского контента
- Редактирование информации о площадках
- Модерация чатов и отзывов
- Управление пользователями (блокировка, предупреждения)

## Инструкция по запуску

1. Скачать архив репозитория и распаковать его
2. Создать базу данных MySQL со следующей структурой:

<img width="836" height="1114" alt="efw" src="https://github.com/user-attachments/assets/aa4629b9-de9d-44ed-95be-3bebcdf14acf" />

3. При необходимости изменить параметры подключения к БД в `DatabaseConnection.java`
4. Собрать проект с помощью `build.bat`
5. Развернуть веб-приложение на Apache Tomcat 9.x
6. Открыть браузер и перейти по адресу `http://localhost:8095/playly/`

## Технологии

### Бэкенд
- **Java** — основной язык программирования
- **Java Servlets & JSP** — обработка HTTP-запросов и рендеринг страниц
- **MySQL** — система управления базами данных
- **JDBC** — подключение к базе данных

### Фронтенд
- **HTML5/CSS3** — верстка и стилизация
- **JavaScript** — клиентская логика
- **Bootstrap** — адаптивный дизайн
- **Leaflet/Яндекс.Карты API** — интерактивные карты
- **JSTL** — серверные шаблоны для JSP
